# 🪨 RockMNG — Employee Management System

**RockMNG** is a modern, responsive web-based employee data management application designed for small to medium-sized organizations. This prototype delivers a fully functional frontend experience with authentication, dashboard analytics, employee CRUD, search/filter, pagination, and report exports — all within a single HTML file.

---

## ✨ Features

- **🔐 Secure Authentication** — Login system with role-based access (admin/manager). Demo credentials included.
- **📊 Dashboard** — Real-time summary cards (total employees, active count, departments, monthly payroll) and recent hires list.
- **👥 Employee Directory** — Full CRUD operations: add, edit, view, and delete employees.
- **🔍 Advanced Filtering** — Search by name/email, filter by department and status, sortable columns.
- **📄 Pagination** — Client-side pagination for large employee lists.
- **📈 Reports & Analytics** — Workforce statistics, department distribution, average salary.
- **📥 Data Export** — Export employee data as CSV or JSON with one click.
- **🎨 Distinct Mineral-Themed UI** — Dark, earthy tones with amber/gold accents, glassmorphism cards, and smooth animations.
- **📱 Fully Responsive** — Optimized for desktop, tablet, and mobile devices.
- **💾 Local Storage Persistence** — All data survives page reloads; pre-loaded with 8 sample employees.
